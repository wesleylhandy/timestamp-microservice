Timestamp microservice

How does this work?

  1. Pass either a date or a unix timestamp as a paramter appended to the url for this page.

  2. If the parameter is either a valid date or a timestamp, this service will return a JSON object that includes both the natural date and the unix timestamp for the requested date.

  3. However, if either parameter is missing, then it will return a JSON object with null values.

    For Instance:

    Input ->

    https://what-time.glitch.me/April%208%201974

    https://what-time.glitch.me/134683200000

    Output ->

    { "unix": 260496000, "natural": "April 4, 1978" }
